# Code quality and formatting

## ruff

- Ultra-fast linter + formatter

```bash
ruff check . # Lint
ruff check --fix . # Auto-fix
ruff format # Format code
```

## black

- Opinionated code formatter

```bash
black . # Format entire project
```

## flake8

- Linting for PEP8 compliance

```bash
flake8 .
```

## isort

- Auto-sort imports

```bash
isort .
```

## mypy

- Static type checker

```bash
mypy .
```
